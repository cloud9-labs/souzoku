# 相続税申告書類管理システム 世界水準リサーチ総合レポート
**作成日: 2026-02-26 | TAISUN World Research**

---

## エグゼクティブサマリー

3エージェント並列リサーチ（相続税法・SaaS/API・UX設計）の統合結果。

### 最大の発見：競合ギャップ
既存の税理士向けソフト（NTTデータ達人・TKC TPS8000等）は**申告書作成**に特化しており、
以下の機能が市場にほぼ存在しない：
- 顧客（相続人）向けポータル（書類アップロード・進捗確認）
- 条件分岐による必要書類の自動生成
- OCR書類自動分類
- 新人税理士向けAIガイダンス
- 段階的なメール・Slack通知

**→ これらを実装することで国内最高水準のシステムが実現できる**

## PART 1: 相続税申告 必要書類 完全マスターリスト

### A. 全案件共通の必須書類

| 書類名 | 取得先 | 費用 | 所要日数 |
|--------|--------|------|---------|
| 被相続人の戸籍謄本（出生〜死亡の全期間） | 本籍地市区町村（令和6年3月〜広域交付可） | 450〜750円/通 | 数日〜2週間 |
| 被相続人の住民票除票 | 住所地市区町村 | 200〜400円 | 数日 |
| 法定相続情報一覧図（任意、戸籍の代替） | 法務局 | 無料 | 1〜2週間 |
| 相続人全員の戸籍謄本 | 各本籍地役場 | 450円/通 | 数日 |
| 相続人全員の住民票 | 各住所地役場 | 200〜400円 | 数日 |
| 相続人全員の印鑑証明書（原本） | 各住所地役場 | 200〜300円 | 数日 |
| 相続人全員のマイナンバー確認書類 | 自宅 | 無料 | 即時 |

### B. 遺産分割状況で分岐する書類

```
遺言書あり？
├─ 公正証書遺言 → 写しをそのまま利用（検認不要）
├─ 自筆証書遺言（法務局保管） → 遺言書情報証明書（検認不要）
├─ 自筆証書遺言（自宅保管） → 原本＋家庭裁判所の検認済証明書
└─ 遺言書なし → 遺産分割協議書＋相続人全員の印鑑証明書

相続放棄者あり → 相続放棄申述受理証明書（家庭裁判所発行）
未分割で申告 → 分割見込書を事前提出（配偶者控除・小規模宅地を後から適用するため）
```

### C. 財産種類別の必要書類（条件分岐）

#### C-1 現金・預貯金
- 残高証明書（死亡日時点）：各金融機関 ※1〜2週間かかる
- 過去5〜7年分の通帳・証書：名義預金調査のため
- 既経過利息計算書：定期預金のみ
- **⚠️見落とし多発：ネット銀行・PayPay・暗号資産**

#### C-2 不動産
- 固定資産税課税明細書（最新年度）
- 名寄帳（不動産所在地ごとに各市区町村で取得）
- 登記事項証明書（法務局 600円/通）
- 公図・地積測量図（法務局 450円/通）
- 賃貸物件あり → 賃貸借契約書・レントロール

#### C-3 上場株式
- 残高証明書（死亡日時点）：各証券会社
- 配当金支払通知書・過去5年分取引明細
- 単元未満株 → 株主名簿管理人（信託銀行）へ照会
- 全口座一括確認 → 証券保管振替機構照会（6,050円）

#### C-4 非上場株式（事業承継）
- 過去3期分の決算書・法人税申告書
- 株主名簿（役職・続柄記載）
- 会社の定款・登記事項証明書
- 子会社がある場合：子会社の財務書類

#### C-5 生命保険・死亡退職金
- 保険金支払通知書（非課税枠：500万円×法定相続人数）
- 保険証書のコピー
- 保険料負担者確認資料（誰が払っていたかで課税関係が変わる）
- 全保険一括確認 → 生命保険協会照会（3,000円）

#### C-6 農地
- 農業委員会の耕作証明書・適格者証明書
- 納税猶予特例申請 → 担保提供書類（登記事項証明書）

#### C-7 債務・葬式費用（控除項目）
- 借入金残高証明書・返済予定表
- 未払医療費・未納税金の領収書・納付書
- 葬儀会社の領収書・明細書（火葬場・霊柩車含む）
- お布施・心付けのメモ（支払日・相手先・金額）

### D. 特例・控除適用時の追加書類

| 特例名 | 主な追加書類 |
|--------|------------|
| 配偶者税額軽減 | 遺産分割協議書＋印鑑証明書（未分割なら分割見込書） |
| 小規模宅地（居住用・配偶者） | 追加不要（最大80%減） |
| 小規模宅地（同居親族） | 住民票・戸籍の附票 |
| 小規模宅地（家なき子） | 過去3年間の賃貸契約書 |
| 小規模宅地（老人ホーム入居） | 介護保険被保険者証＋入居契約書 |
| 相続時精算課税 | 過去の贈与税申告書（全年分）＋戸籍の附票 |
| 未成年者控除 | 戸籍謄本（年齢確認）控除額：(18歳−年齢)×10万円 |
| 障害者控除 | 障害者手帳コピーまたは医師診断書 |
| 農地納税猶予 | 農業委員会発行適格者証明書＋担保提供書類 |

## PART 2: 申告タイムライン（期限管理の全体像）

```
死亡当日
├─ 7日以内：死亡届の提出・火葬許可証
│
├─ 1ヶ月以内（最優先）
│   ├─ 遺言書確認・検認手続き申立て
│   ├─ 連続戸籍謄本の収集開始 ← 全手続きの基礎
│   ├─ 各金融機関への残高証明書発行依頼
│   └─ 税理士への相談・委任
│
├─ 3ヶ月以内（法定期限）
│   ├─ 相続の承認または放棄の申述
│   └─ 財産目録の作成
│
├─ 4ヶ月以内（法定期限）
│   └─ 準確定申告（被相続人の所得税・消費税）
│
├─ 5〜7ヶ月
│   ├─ 遺産分割協議の開始
│   ├─ 不動産評価（路線価：毎年7月公表）
│   ├─ 非上場株式の評価計算
│   └─ 特例適用書類の収集
│
├─ 8〜9ヶ月
│   ├─ 遺産分割協議書の完成（目安）
│   ├─ 相続税申告書の作成・確認
│   └─ 納税資金の確保
│
└─ 10ヶ月以内（申告・納税期限）
    ├─ 相続税申告書の提出（被相続人住所管轄の税務署）
    └─ 相続税の納税
```

**期限違反ペナルティ：**
- 無申告加算税：15%（自主申告なら5%、50万円超は20%）
- 延滞税：年8.7%前後（令和6年時点）
- 重加算税（隠ぺい・仮装）：35〜40%

### ヒアリングページ 5ステップフロー

**UXの核心原則：**
- 死亡日入力と同時に「申告期限：〇年〇月〇日」を自動表示
- 子の人数入力と同時に「基礎控除額：〇〇〇〇万円」をリアルタイム更新
- 財産スクリーニングはスイッチカード型（ONにした項目のみ次ステップで詳細入力）
- 表現への配慮：「死亡日」→「ご逝去日」、「子供」→「お子様」

```
STEP 1: 被相続人情報（必須）
  └─ 氏名・生年月日・ご逝去日・最終住所・本籍地・国籍

STEP 2: 相続人の確定
  └─ 配偶者有無・お子様人数・養子有無・代襲相続人有無
      遺言書有無（公正証書/自筆/なし）・相続放棄者有無

STEP 3: 財産概況スクリーニング（スイッチカード型）
  └─ 不動産 / 農地・山林 / 預貯金 / 上場株式
      非上場株式（事業） / 生命保険 / 死亡退職金
      海外資産 / 暗号資産・電子マネー / その他

STEP 4: 特例・特殊事情
  └─ 小規模宅地の特例（居住用/事業用/貸付用）
      配偶者税額軽減 / 生前贈与（7年以内/精算課税）
      相次相続控除 / 障害者控除 / 未成年者控除
      延納・物納の検討

STEP 5: 書類収集の現状確認
  └─ すでに手元にある書類にチェック
      → プログレスバーが即時更新「あとN点で基本書類が揃います」
```

### 書類チェックリスト 3層構造

```
【TIER 1】必須書類（全案件共通）
  □ 被相続人の戸籍謄本（出生〜死亡）  ← 入手先・費用・日数を表示
  □ 被相続人の住民票除票
  □ 相続人全員の戸籍謄本
  □ 相続人全員のマイナンバー確認書類

【TIER 2】財産種類別書類（ヒアリングSTEP3の回答から自動生成）
  ▼ 不動産あり の場合に表示
  □ 固定資産税課税明細書
  □ 名寄帳（〇〇市・〇〇市）
  □ 登記事項証明書（法務局）
  ...

【TIER 3】特例・控除関連書類（ヒアリングSTEP4の回答から自動生成）
  ▼ 小規模宅地の特例（家なき子）を申請する場合
  □ 過去3年間の賃貸契約書
  □ 同居していなかったことの証明書類
  ...
```

**各書類カードのUI：**
```
┌─────────────────────────────────────────┐
│ [□] 固定資産評価証明書                      │
│ 入手先: 市区町村役場（各不動産の所在地）      │
│ 取得日数の目安: 即日〜1週間                  │

│ ⚠️ 注意: 相続開始年度のものが必要            │
```

### 通知フロー設計（3段階エスカレーション）

```
ご逝去日
  +3日   【税理士内部】受任確認・初回連絡
  +7日   【顧客へ】書類依頼メール（TIER1、柔らかいトーン）
  +30日  【顧客へ】第1回進捗確認（未提出書類リスト自動添付）
  +60日  【内部アラート】収集率60%未満→担当者にSlack通知
  +90日  【顧客へ】第2回リマインダー「申告期限まで残り7ヶ月」
  +240日 【顧客へ】第3回リマインダー「残り30日・高緊急度」
           同時→担当税理士＋所長へエスカレーション通知
  +270日 【申告期限】
  期限後  ペナルティ説明・更正の請求フロー案内
```

**メール文面の原則：**
- 第1回：「ご逝去に際し謹んでお悔やみ申し上げます」から始まる柔らかいトーン
- 第3回：「残り30日」「未提出書類N点」を明確・簡潔に伝える

### 新人税理士向けガイダンス機能

1. **案件受任時のAI解析パネル**
```
【この案件のポイント（AI解析）】
・不動産が3筆あります
  → 「小規模宅地の特例」の適用可否を確認してください
  [小規模宅地の特例について詳しく見る]
・非上場株式が含まれています
  → 法人税申告書3期分の入手に2〜3週間かかります
  [事業承継税制について詳しく見る]
・申告期限まで287日（2025年9月15日）
  [標準タイムラインを確認する]
```

2. **提出前ミス防止チェッカー（自動実行）**
```
✓ 戸籍謄本の期間連続性OK（出生日〜死亡日）
✗ 警告：非同居の配偶者がいる相続人あり
  「家なき子特例」の適用条件（3年内持ち家なし）を確認
✓ 法定相続人数と基礎控除額が一致（3人・4,800万円）
✓ 生命保険の非課税枠正しく計算（500万×3人=1,500万円）
✗ 警告：農地の納税猶予申請しましたが農業委員会証明書が未提出
```

3. **案件完了後のナレッジ蓄積**
   - 担当税理士が「このケースで困ったこと・注意点」を登録
   - 類似案件で自動サジェスト

## PART 8: 法的注意事項（最重要）

### 税理士法との境界線
- **適法：** 書類収集・管理・進捗確認・チェックリスト提供
- **グレーゾーン：** 税額の概算計算（「参考値」として明記が必要）
- **違法（税理士資格が必要）：** 税務申告書の自動作成・税務判断

**→ 推奨設計：** 「書類収集・管理・税理士への引き渡しプラットフォーム」として位置付け、申告書作成は提携税理士が行う構成
